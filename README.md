ste projeto é um script automatizado desenvolvido em **Python** para gerar um currículo profissional de alta qualidade em formato PDF. Ele utiliza a biblioteca `ReportLab` para construir um layout moderno, limpo e estruturado, ideal para profissionais de tecnologia.

## 🚀 Funcionalidades
- **Geração Dinâmica:** O PDF é gerado em segundos a partir de dados em Python.
- **Layout Moderno:** Cabeçalhos estilizados, separadores visuais e tipografia profissional.
- **Customização Simples:** Variáveis centralizadas para fácil edição de experiências, formação e contatos.
- **Formato A4:** Padronizado para sistemas de recrutamento e impressão.

## 🛠️ Tecnologias
- [Python 3.14+](https://www.python.org/)
- [ReportLab](https://www.reportlab.com/) (Motor de geração de documentos)

## 📦 Instalação e Uso

1. **Clone o repositório:**
   bash:
   git clone [https://github.com/fabricio-sec/script_curriculo.git](https://github.com/seu-usuario/nome-do-repo.git)
   cd nome-do-repo

2. **Instale as dependências**
bash:
    pip install -r requirements.txt

3. Configure seus dados:
    Abra o arquivo curriculoSCRIPT2.py e edite as constantes no bloco de DADOS PESSOAIS, EXPERIÊNCIA e ESCOLARIDADE.

Gere o PDF:

Bash:
    python curriculoSCRIPT2.py
O arquivo curriculo.pdf aparecerá na pasta raiz.
